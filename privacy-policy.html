Privacy Policy for Subslator

Last updated: March 2026

info 1. Overview

Subslator is a Chrome extension designed to help users understand and translate subtitles directly while watching videos. This Privacy Policy explains what data we collect, how we use it, and how we protect your information.

We are committed to ensuring that your privacy is protected. Subslator is built with a privacy-first approach and collects only the minimum data necessary to provide its core functionality.

visibility_off 2. Information We Collect

2.1 No Personal Data Collection

Subslator does not collect, store, or share any personally identifiable information (PII) such as:

  • Name
  • Email address
  • Phone number
  • Location data
  • Account credentials

2.2 Text Processing (User-Selected Content)

When you use Subslator:

  • The extension processes text that you select (e.g., subtitle words or phrases)
  • This text may be sent to a translation API to provide:
    • Translation
    • Word meaning
    • Pronunciation
    • Definitions
👉 This processing is done only when you actively select text.

2.3 Automatically Collected Data

Subslator does not automatically collect:

  • Browsing history
  • Keystrokes
  • Page content (except selected text)
  • Cookies or tracking data

memory 3. How We Use Information

We use the selected text solely to:

  • Detect the source language
  • Translate content into the user’s selected language
  • Provide word meanings, pronunciation, and definitions
  • Improve the accuracy and performance of translations

api 4. Third-Party Services

Subslator may use third-party APIs for:

  • Language detection
  • Translation
  • Dictionary data

These services may temporarily process the selected text to return results. We do not control how these services handle data, so we recommend reviewing their respective privacy policies.

However:

  • We do not store or log any text sent to these services
  • Data is used only for real-time response

storage 5. Data Storage

Subslator stores minimal data locally in your browser, such as:

  • User preferences (e.g., selected target language)
  • UI settings

This data:

  • Never leaves your device
  • Is not shared with any external server

share_off 6. Data Sharing

We do not sell, trade, or share your data with third parties.

Your data is only shared when necessary to:

  • Process translations via third-party APIs (as described above)

shield 7. Security

We take reasonable measures to protect your data:

  • No sensitive data is stored
  • Communication with APIs is done over secure connections (HTTPS)

tune 8. Your Control

You are always in control:

  • Subslator only works when you select text
  • You can disable or uninstall the extension at any time
  • You can clear stored preferences from your browser settings

child_care 9. Children's Privacy

Subslator is not intended for children under the age of 13. We do not knowingly collect any data from children.

update 10. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated date.
